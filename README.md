# OpenAI-Powered AI Chatbot

## Overview
A modular Python chatbot that combines rule-based intent detection with OpenAI-powered responses.

## Features
- OpenAI GPT-based response generation
- Conversation memory
- Modular architecture
- Resume-ready project

## Setup
```bash
pip install -r requirements.txt
export OPENAI_API_KEY=your_key_here
python app.py
```

## Why This Project
Demonstrates real-world LLM integration, clean backend design, and scalable AI systems.
